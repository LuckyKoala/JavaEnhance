## 题目

> 4. 二维数组中的查找

给定一个二维数组，其每一行从左到右递增排序，从上到下也是递增排序。给定一个数，判断这个数是否在该二维数组中。

```
对于下面的二维数组:
[
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

如果查找 5, 返回 true.
如果查找 20, 返回 false.
```

## 解法

一开始使用二分查找，发现情况很复杂，没解出来。

按照《剑指Offer》中的解释，代入参数找规律，解法是从右上角或左下角开始查找，若是不等于，则根据情况减少一行/一列的搜索范围，这样搜索范围越来越小就可以确定最终结果了。
